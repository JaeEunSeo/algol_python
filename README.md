# 📜 Commit Convention

명확한 학습 로그를 기록하기 위해 다음과 같은 커밋 컨벤션을 따릅니다.

---

## 🛠️ 기본 형식

### 예시

- `[BOJ] 1000 A+B 문제풀이`
- `[Programmers] 42576 완주하지 못한 선수 문제풀이`
- `[LeetCode] 1 Two Sum 문제풀이`

---

## ✍️ 플랫폼 태그

| 플랫폼       | 태그          |
| ------------ | ------------- |
| 백준         | `BOJ`         |
| 프로그래머스 | `Programmers` |
| 리트코드     | `LeetCode`    |
| 기타         | `Etc`         |
